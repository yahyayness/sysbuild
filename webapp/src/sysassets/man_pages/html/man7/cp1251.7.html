NAME

cp1251 - CP 1251 character set encoded in octal, decimal, and hexadecimal  

DESCRIPTION

The Windows Code Pages include several 8-bit extensions to the ASCII character set (also known as ISO 646-IRV). CP 1251 encodes the characters used in Cyrillic scripts. The following table displays the characters in CP 1251, which are printable and unlisted in the ascii(7) manual page.
